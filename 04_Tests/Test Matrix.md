
| Test                  | Purpose                                     | Method                                                       | Success Criteria                     |
| --------------------- | ------------------------------------------- | ------------------------------------------------------------ | ------------------------------------ |
| Sun Tracking Accuracy | Verify automatic sunlight detection         | Move light source to different directions                    | Panel rotates toward strongest light |
| Dual-Axis Motion      | Verify two-axis adjustment                  | Test horizontal and vertical servo movement                  | Smooth motion without interference   |
| Charging Output       | Verify outdoor charging function            | Measure voltage/current from solar panel and charging module | Stable charging output               |
| Remote Control        | Verify manual user control                  | Send Wi-Fi/Bluetooth commands from phone                     | System responds correctly            |
| Mechanical Stability  | Verify frame and joints are stable          | Run repeated movement cycles and apply small disturbance     | No slipping, loosening, or failure   |
| Safety / Fail-Safe    | Verify safe behavior under error conditions | Block sensors or test abnormal input                         | System stops or enters safe state    |
