| Test Category                 | Test Objective                                                                  | Test Method                                                                                                 | Success Criteria                                                               | Parts Tested                                     |
| ----------------------------- | ------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------ |
| Sun Tracking Accuracy         | Verify that the system can detect sunlight direction and adjust the panel angle | Place a light source at different positions and record whether the panel rotates toward the strongest light | Panel moves toward the strongest light source within an acceptable angle range | LDR sensors, ESP32, servo motors, panel frame    |
| Dual-Axis Movement            | Verify horizontal and vertical adjustment                                       | Command the panel to rotate left/right and up/down; observe range of motion                                 | Both axes move smoothly without collision or unstable motion                   | Two servos, joints, bracket                      |
| Outdoor Charging Output       | Verify that the solar panel can provide usable charging power                   | Place the panel under sunlight and measure output voltage/current before and after charging module          | Charging module provides stable output suitable for battery or device charging | Solar panel, charging module, battery/USB output |
| Remote Control                | Verify user can control the system remotely                                     | Send commands through Wi-Fi or Bluetooth, such as manual left/right/up/down or auto/manual mode switch      | ESP32 receives commands and responds correctly                                 | ESP32, phone/computer interface, servo motors    |
| Power Consumption             | Check whether the system can operate without using excessive power              | Measure current draw of ESP32 and servo motors during tracking                                              | System power demand does not exceed available power from battery/solar system  | ESP32, servos, battery                           |
| Mechanical Stability          | Verify that the panel and frame remain stable during movement                   | Run repeated servo movement cycles and observe looseness, vibration, or slipping                            | No major looseness, slipping, or structural failure after repeated cycles      | Frame, joints, servo mounts, panel mount         |
| Wind / Disturbance Resistance | Check whether the panel can tolerate small outdoor disturbances                 | Apply light manual force or airflow to the panel and observe if it stays mounted and functional             | Panel remains attached and returns to controlled position                      | Panel, bracket, base, joints                     |
| Weather Exposure Protection   | Check whether electronics are protected for outdoor use                         | Inspect enclosure and wiring protection; optionally expose to light splash or simulated outdoor condition   | Electronics remain covered; no exposed unsafe wiring                           | ESP32 housing, wires, charging module enclosure  |
| Mode Switching                | Verify smooth transition between automatic and manual control                   | Switch between auto tracking and remote manual control                                                      | System changes mode without unexpected motion or reset                         | ESP32 software, sensors, remote interface        |
| Safety / Fail-Safe            | Verify system behavior under sensor error or obstruction                        | Cover one or more LDR sensors or disconnect sensor input during test                                        | System stops, holds position, or enters safe/default behavior                  | ESP32 code, sensors, servo motors                |
